# About This Project

This is a fan-made English translation and reference wiki for the Union Arena card game, maintained by a single translator/editor.

## Goals

- Provide accurate, readable English translations of card text and rules
- Serve as a reference for English-speaking players and collectors

## Roadmap

- [x] Basic wiki structure
- [ ] Full card database with search and filter
- [ ] Community contributions (planned for a future version)
